# CA Monk Blog Application

A simple blog application built as part of the **CA Monk Frontend Assignment**.  
Users can create blogs, view a list of blogs, and read detailed blog content.

---

## Features

- Create a blog with:
  - Title
  - Description
  - Image URL
  - Content
  - View list of all blogs
  - View blog details with:
  - Category
  - Published date
  - Image
  - Full content
  - Clean and responsive UI

---

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- TanStack Query
- JSON Server (mock backend)
